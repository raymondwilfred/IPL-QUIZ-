# IPL-QUIZ-
An interactive command-line quiz game based on the Indian Premier League (IPL). Test your knowledge about IPL records, players, captains, teams, and achievements!

IPL Quiz Game | Python

Developed an interactive command-line quiz application using Python.

Implemented randomized question selection using the random module.

Designed modular functions for question handling and score tracking.

Utilized lists and dictionaries for efficient data storage.

Improved user interaction through input validation and result display.
